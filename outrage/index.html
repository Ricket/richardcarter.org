Outrage!

To play the game, run this file (preferably from Internet Explorer).

Instructions: After clicking the Play button, click and drag your mouse downward and let go to send the dog flying (the original idea was to pull the dog's tail and let go). Hold down the spacebar to make the dog stick to the bouncy blocks. Try to collect all of the items in each level!

If you can't load the game because your .NET version is outdated or something, then use this setup.exe file, which will install anything your computer might need along with the game.

If you're on Mac or Linux, you're out of luck. Install Windows.
